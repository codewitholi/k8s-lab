## Installation

```bash
python3 -m pip install --user pipx
python3 -m pipx ensurepath

exec "$SHELL"

pipx install hatch
```

Activate the environment, install Ansible.

```bash
$ hatch shell
```

Use *CTRL+D* to exit hatch shell.
